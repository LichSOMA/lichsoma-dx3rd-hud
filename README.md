# LichSOMA's DX3rd HUD

Foundry VTT의 Double Cross 3rd Edition 시스템을 위한 HUD 모듈입니다.

## 기능

- **플레이어 HUD**: 플레이어 캐릭터의 상태를 화면에 표시
- **에너미 HUD**: 적 캐릭터의 상태를 화면에 표시
- **커스텀 전투 추적기**: DX3rd에 최적화된 전투 순서 추적기
- **승화 컷인**: 캐릭터 승화 시 화려한 컷인 애니메이션
- **턴 강조 애니메이션**: 현재 턴인 캐릭터의 토큰 강조
- **플레이어 활동 감시**: 플레이어 휴면 상태 알림 기능
- **턴 알림**: 플레이어의 턴이 되었을 때 소리와 메시지로 알림

## 설치 방법

1. 압축 파일을 다운로드합니다
2. Foundry VTT의 `Data/modules` 폴더에 압축을 풉니다
3. Foundry VTT를 재시작하고 월드에서 모듈을 활성화합니다

## 사용법

### 기본 설정

1. 게임 세계에 접속 후 **Game Settings** 메뉴 열기
2. **Module Settings** 탭에서 "LichSOMA's DX3rd HUD" 찾기
3. 원하는 설정 조정:
   - HUD 이름 폰트
   - 전투 추적기 표시 칸 수
   - 턴 강조 애니메이션 설정
   - 턴 알림 사운드 및 메시지
   - 플레이어 휴면 상태 감시 활성화

### HUD 활성화

- **플레이어 HUD**: 화면 좌측 상단의 토글 버튼 클릭
- **에너미 HUD**: 화면 우측 상단의 토글 버튼 클릭
- 전투 시작 시 자동으로 활성화되도록 설정 가능

### 캐릭터 이미지 설정

1. 캐릭터 시트 열기
2. **HUD 이미지 설정** 버튼 클릭
3. 이미지 경로 입력 및 위치/크기 조정
4. 미리보기로 확인 후 저장

### 전투 추적기 사용

1. 전투 시작 시 자동으로 화면 하단에 표시
2. 현재 턴인 캐릭터가 중앙에 강조 표시됨
3. GM은 캐릭터를 더블클릭하여 해당 플레이어에게 턴 알림 전송 가능
4. 우클릭 메뉴로 행동 종료 해제, 우선권 재확인 등 가능

### 승화 컷인

- DX3rd 시스템에서 승화를 사용하면 자동으로 컷인 애니메이션 재생
- 설정에서 승화 컷인 사운드와 볼륨 조정 가능

## 필요 사항

- Foundry VTT v13 이상
- Double Cross 3rd Edition 시스템
